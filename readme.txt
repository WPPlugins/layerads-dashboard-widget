This Widget shows the current status of your layer-ads.de-account in the Dashboard of Wordpress.

== Description ==

Every Account at Layer-Ads.de gets an XML-URL with a lot of Statistics. This Plugin integrates this data into the Dashboard of Wordpress.

== Installation ==

1. Upload `layerads-dashboard-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your Dashboard
4. Edit the Widget
5. Login to your Layer-Ads.de-Account and go to "Statistiken".
6. At the bottom of this page you can find the XML-URL which begins with http://layer-ads.de/api/
7. Copy the Part after "http://layer-ads.de/api/" into the corresponding Field.
8. If needed, edit the Userlevel for accessing the Widget.

== Frequently Asked Questions ==

= How can i get an Account at Layer-Ads.de =

Click [here](http://layer-ads.de/refer.php?60116 "Get an Account at Layer-Ads.de")

= Where can i get the XML-URL of my Account? =

Login to your Account and Click on "Statistiken". You can find the XML-URL at the bottom of this Site.
